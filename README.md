# StayWorld – Complete Website

## 포함된 페이지
- `index.html` – 메인 및 전체 메뉴
- `login.html` – 로그인 폼
- `signup.html` – 회원가입 폼
- `reservation.html` – 예약 시스템
- `subscribe.html` – 유료 구독 페이지
- `style.css` – 모든 페이지 공통 스타일 시트

## 배포 방법
1. GitHub Pages 또는 Netlify 업로드
2. index.html이 루트로 위치해야 함
3. 연결된 하위 페이지도 자동 인식됨
